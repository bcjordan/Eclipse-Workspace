About the Vocab List Generator

The vocab list generator is a tool that can be used to transform lists of untranslated words into lists with translations.

Using a flash card tool like quizlet or supermemo, you can then import these hyphen-separated terms to generate cards to study with.

The Vocab List generator was programmed using GWT, GWT Language for translation, forvo for human pronunciations and flickr for a random background image.

Audio pronunciations require a real browser with HTML5 audio.
